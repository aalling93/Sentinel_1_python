PACKAGE CONTENT
===============

This package contains OpenStreetMap data of the
coastline land polygons, simplified for rendering at low zooms.

Layers contained are:

simplified_land_polygons.shp:

  63530 Polygon features
  Mercator projection (EPSG: 3857)
  Extent: (-20037507, -20037507) - (20037508, 18461504)
  In geographic coordinates: (-180.000, -85.051) - (180.000, 83.666)

Date of the data used is 2022-06-06T00:00:00Z

You can find more information on this data set at

https://osmdata.openstreetmap.de/data/land-polygons.html
